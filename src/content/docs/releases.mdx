Stay up to date with the latest updates, improvements, and new features in Trio Agent. This page lists all major releases and their highlights.

## v1.0.0 (Initial Release)
- Launch of Trio Agent with core features:
  - Massive 2M token context window
  - Local-first, private, no lock-in
  - Cross-platform (Windows, Mac, Linux)
  - Bring your own API keys (OpenAI, Gemini, Claude, etc.)
  - GitHub integration
  - Supabase integration
  - Real-time collaboration
  - Advanced analytics
  - Customizable themes and dark mode
  - Offline mode
  - User feedback system
  - Automated workflows
  - Enhanced security features
  - Comprehensive tutorials and guides
