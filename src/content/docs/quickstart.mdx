# Quickstart

Learn how to set up Trio Agent and start building your first app.

## Installing Trio Agent
1. **Download Trio Agent** for your platform (Windows, Mac, Linux) from the official site.
2. Open the app. On Windows, you may see a blue warning from Microsoft—click "More info" then "Run anyway."

## Setup Steps
When you open Trio Agent, you'll see a banner on the home screen indicating  Trio AI ready no setup needed, You can nw start building and experimenting Trio Agent featres and functionlities for free. In addition if you want to setup and use a differnt provider, you can typically do that by clicking settings in sidebar, go under AI and configure your prefered provider with your api key.

### 1. Install Node.js
Trio Agent requires Node.js to run your apps. Download and install it from the [official Node.js site](https://nodejs.org/).

### 2. Setup AI Access
You’ll typically want to access AI models via APIs like Google Gemini, Anthropic Claude, or OpenAI.

**Recommended:** Create a Google Gemini API key for a generous free tier (e.g., 250 messages/day for Gemini 2.5 Flash).

## Creating Your First App
1. Type something into the chat input and click send (or press Enter).
2. Trio Agent will start building an app for you. After the AI finishes generating the code, you'll need to approve it.
3. Once approved, the code is saved and run on your computer. You’ll see a preview of the app open on the right side.
