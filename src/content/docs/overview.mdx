


# Overview

Welcome to Optima/Trio Agent documentation!

This guide will help you get started, integrate, and master the platform.

## What is Optima/Trio Agent?

Optima/Trio Agent is a next-gen, local-first AI coding agent and app builder. It offers:

- Massive context window (2M tokens)
- Local-first, private, no lock-in
- Cross-platform (Windows/Mac/Linux)
- Real-time collaboration & analytics
- App builder, templates, and integrations
