
# GitHub Integration

Trio Agent has built-in integration with GitHub, making it easy to push your local changes to a GitHub project.

## How to Sync
1. Open the app details card in Trio Agent and click **Connect to GitHub**.
2. Authorize access as prompted.
3. Click **Sync to GitHub** to push your code from your local computer to GitHub. You can do this anytime to push new changes.

## Advanced Workflows
- Use **GitHub Desktop** for a user-friendly interface.
- Use the **Git CLI** for advanced workflows.

For step-by-step instructions, see the [Publishing your App](/docs/publishing) guide.
